# TrankaSoft Landing

Landing page de [trankasoft.com](https://trankasoft.com) que presenta TrankaPOS y deriva al producto en `pos.trankasoft.com`.

## Stack

- Vite 8 + React 19 + TypeScript 6
- Tailwind CSS 4 (vía `@tailwindcss/vite`)
- lucide-react para iconos

## Desarrollo

```bash
npm install
npm run dev
```

Servidor en http://localhost:5173

## Build

```bash
npm run build
npm run preview
```

## Deploy

Vercel detecta el proyecto Vite automáticamente. Production branch: `main`.
